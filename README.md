# dotfiles

## Quick Start

```sh
curl -fsSL https://raw.githubusercontent.com/jruhland/dotfiles/main/bootstrap.sh | sh
```

## What Gets Installed

- Homebrew + packages (see [Brewfile](./Brewfile))
- Oh My Zsh
- asdf (nodejs 24.x, python 3.14.x)
- Neovim, Ghostty, 1Password, etc.
- [macOS only] System preferences via macos.sh
